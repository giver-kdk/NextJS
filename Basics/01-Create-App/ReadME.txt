- To install NextJS framework, use:
	npm i next -g 
- To create next app, use: 
	npx create-next-app
- To run next app in dev, use: 
	npm run dev
- On app start, control flows from package.json -> _app.js -> index.js
- "public" folder contains all the public resources(not index.html)
- "styles" folder contains all the css
